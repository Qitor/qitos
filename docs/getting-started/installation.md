# Installation

## Requirements

- Python 3.10+

## Install QitOS (editable)

```bash
pip install -e .
```

## Docs dependencies (optional)

```bash
pip install -r docs/requirements.txt
```

## Run the docs locally

```bash
mkdocs serve
```
